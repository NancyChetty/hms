# Amazon Q — Debugging Assistance

## Topics Covered
- PostgreSQL terminal issues
- Path configuration
- Migration debugging
- Serverless runtime fixes

## Example Prompts

### Prompt
Why is psql command not recognized?

### Response Summary
- PostgreSQL PATH missing
- Use PostgreSQL bin directory
- Add environment variable

---

### Prompt
Why are migrations failing?

### Response Summary
- Missing columns
- Stale migrations
- Database schema mismatch

---

### Prompt
Why is serverless offline not starting?

### Response Summary
- Install serverless globally
- Install serverless-offline plugin
- Verify Node.js version
